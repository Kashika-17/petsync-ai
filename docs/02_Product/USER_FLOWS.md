# Universal User Experience Rules

- Maximum three clicks to reach any major feature.
- Every page has one primary action.
- Inline editing wherever possible.
- Auto-save for supported forms.
- Undo for destructive actions.
- Loading states for every async operation.
- Skeleton screens instead of blank pages.
- Clear success and error feedback.
- Keyboard accessible navigation.
- Responsive across desktop, tablet, and mobile.
